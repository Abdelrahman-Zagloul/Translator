##  Features
- Integrates with **OpenAI**.
- Translate text to multiple languages using **OpenAI GPT API**.
- Retrieve a list of supported languages.
- Minimal API design for simplicity.
- CORS enabled for cross-origin requests.
---

## How to Run

1. **Clone the repository**  
   ```bash
   git clone https://github.com/Abdelrahman-Zagloul/Translator.git
2. Add your `appsettings.json` (not included in repo):

   ```json
     "OpenAI": {
      "Key": "YOUR_API_KEY"
      }
   ```
3. Run the project:

   ```bash
   dotnet run
   ```
